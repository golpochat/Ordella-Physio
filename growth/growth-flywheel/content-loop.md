## Case Studies

### Case study pipeline

```
Customer NPS ≥ 8 → CS asks permission → Interview → Draft → Legal review → Publish → Sales asset
```

### Template structure

1. **Clinic profile** — size, location, prior tools
2. **Challenge** — pain points (scheduling, notes, billing)
3. **Solution** — Ordella features used
4. **Results** — quantified outcomes (placeholder metrics)

# TLS certificates

Place custom TLS certificates here for local or production use.

## Files

| File | Purpose |
|------|---------|
| `acme.json` | Auto-generated by Traefik ACME (Let's Encrypt). **Do not commit.** |
| `ordella.local.crt` | Optional local development certificate |
| `ordella.local.key` | Optional local development private key |
| `.htpasswd` | Optional basic-auth users file for the Traefik dashboard |

## Let's Encrypt (production)

1. Set `ACME_EMAIL` and `ACME_DOMAIN` in `.env`.
2. Ensure ports 80 and 443 are reachable from the internet.
3. Traefik stores issued certificates in `acme.json` automatically.

## Local development (self-signed)

```bash
openssl req -x509 -nodes -days 365 -newkey rsa:2048 \
  -keyout ordella.local.key \
  -out ordella.local.crt \
  -subj "/CN=ordella.local"
```

## Dashboard basic auth

```bash
htpasswd -nb admin your-password > .htpasswd
```
